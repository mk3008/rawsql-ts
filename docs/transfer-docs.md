# Transfer Docs

Transfer Docs are generated review views for the `@rawsql-ts/transfer` logical model and DDL.

The source of truth lives under `packages/transfer/`.
These pages are regenerated by `pnpm docs:transfer` and by the GitHub Pages build.

## What Is Transfer?

`@rawsql-ts/transfer` は、データソース、転送先テーブル、変更検知記録、転送実行、監査向けの由来追跡を定義するための転送制御パッケージです。

このパッケージは、長く使える論理ドキュメントを起点に設計しています。

- Concept Specs は、安定した用語、責務、境界、不変条件を定義します。
- DFDs は、業務とその入力・出力となる概念を定義します。
- Process Maps は、業務ロジックが複雑で手順化が必要な場合にだけ、詳細な処理フローを定義します。
- DDL docs は、概念とプロセスを物理化するテーブル定義を示します。

これらのページはレビュー補助用の生成ビューです。修正する場合は、`packages/transfer/` 配下の Concept Specs、DFD metadata、Process Maps、DDL、table-doc metadata を編集してください。

## Review Order

Use these pages to review the transfer package from stable logical intent toward physical design.
Start with Concepts, then DFDs, then Processes, and finally Table Definitions when reviewing implementation-facing structures.
