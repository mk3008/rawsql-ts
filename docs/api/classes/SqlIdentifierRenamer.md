Handles safe renaming of SQL identifiers within plain SQL strings.

## Example

```typescript
const renamer = new SqlIdentifierRenamer();
const sql = 'SELECT u.id FROM users u';
const result = renamer.renameIdentifier(sql, 'u', 'users_alias');
```
